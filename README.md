# E-Commerce Web Application

## Features
- User Authentication (JWT)
- Product Management
- Shopping Cart
- Checkout System
- Admin Dashboard

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB

## How to Run
1. Go to backend folder → npm install → npm start
2. Go to frontend folder → npm install → npm run dev
